# Lacuna (web)

Next.js app for Lacuna. See the [repository README](../README.md) for the full problem statement, features, and credits.

## Quick start

From the repository root:

```bash
npm install --prefix web
npm run dev
```

Or from this directory:

```bash
npm install
npm run dev
```

Copy `web/.env.example` to `web/.env.local` and set `GEMINI_API_KEY` if you use **Admin → Run analysis flow**.
